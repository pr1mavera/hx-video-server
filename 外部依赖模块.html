外部依赖模块

华夏视频服务 SDK 文档

为了更灵活的可配置性和更优的热更新性能，提供一个全局的依赖容器，供第三方注册
在小程序中初始化sdk时注入依赖，sdk内部运行时自动捕获到依赖进行使用

容器是一个简单的对象类型，提供存取机制（getDependence / updateDependence）

# API

# getDependence

  • @desc - 获取当前外部依赖集合
  • @param - 无
  • @return - Object

# updateDependence

  • @desc - 更新内部依赖
  • @param dep:any - 新的依赖，若传入对象，则会使用解构与之前依赖合并，否则直接覆盖依赖
  • @return - Object
